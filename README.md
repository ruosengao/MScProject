# Stochastic Simulation Applied to Elliptic Equations

<!-- USAGE -->
## Usage
Change directory:
```sh
$ cd code
```
Show help message:
```sh
$ python3 -m eosim -h
```
Run simulations:
```sh
$ python3 -m eosim oop-parallel exit-time config/exit_time_config.json
```
```sh
$ python3 -m eosim oop-parallel occupation-time config/occupation_time_config.json
```
